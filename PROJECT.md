# auto-band-selector

## 프로젝트 개요
*   **제작자:** sleepysoong (hyunnn1123@naver.com)
*   **목적:** 데이터 속도가 느린 특정 위치(군대 생활관 등)에서 스마트폰이 혼잡한 주파수(예: B5)를 자동으로 잡는 문제를 해결하기 위해, 삼성 갤럭시에 내장된 히든 네트워크 설정 메뉴를 조작하여 쾌적한 주파수(예: B1, B7)로 수동 고정하는 과정을 자동화합니다.
*   **작동 방식:** 안드로이드 AccessibilityService(접근성 서비스) API를 이용하여 화면의 UI 노드를 인식하고 매크로 터치/클릭을 수행하여 밴드 설정을 변경합니다.

## 개발 환경 및 기술 스택
*   **언어:** Kotlin
*   **최소 SDK:** API 26 (Android 8.0)
*   **타겟 SDK:** API 34 (Android 14)
*   **주요 API:**
    *   `AccessibilityService`: 화면의 텍스트 노드 파악 및 자동 터치/스와이프 (핵심 매크로 엔진)
    *   `Intent.ACTION_DIAL`: 히든 메뉴(`*123456#`, `319712358`) 진입을 위한 다이얼러 호출

## 작동 단계 (매크로 시나리오)
1.  **실행:** 앱 메인 화면에서 버튼을 누르면 다이얼러 앱이 켜지며 `319712358`이 입력됨.
2.  **비밀번호 입력:** 통신사별 히든 메뉴 비밀번호(SKT: 996412, KT: 774632, U+: 0821)를 접근성 서비스가 자동 입력.
3.  **메뉴 이동:** `Network Settings` -> `Network mode` -> `Band Selection` 순서로 자동 클릭.
4.  **밴드 고정:** 기존에 선택된 `Automatic` 또는 혼잡 밴드를 체크 해제하고, 사전에 지정된 쾌적한 밴드(예: B1, B7)를 찾아 체크 후 `SELECTION` 버튼 터치.
5.  **종료:** 조작이 끝나면 앱 메인 화면(또는 홈 화면)으로 자동 복귀.
